#  SushiTech

---

## <a name="português"></a>🇧🇷 Português

O **SushiTech** é um site digital desenvolvido para o setor gastronômico oriental. A interface combina uma estética minimalista com uma navegação intuitiva, oferecendo um catálogo de produtos fluido e totalmente responsivo para todos os dispositivos.

###  Desenvolvimento com IA
Este projeto é desenvolvido de forma colaborativa com **IA (Claude & Cursor)**. Utilizo essas ferramentas para:
* Acelerar a prototipagem de componentes React.
* Refinar a lógica de programação e arquitetura de pastas.
* Aprender boas práticas de código em tempo real.

###  Tecnologias
* **React** + **Vite**
* **JavaScript (ES6+)**
* **Lucide React** (Ícones)
* **GitHub Pages** (Hospedagem e Deploy)
